# Task 2: Python Recap Assignment

## Objective

This project reads student submission data from a CSV file and generates a summary report in JSON format.

## Features

- Reads student data from CSV
- Calculates total students
- Calculates submitted and missing submissions
- Calculates average score
- Finds highest and lowest scorers
- Calculates domain-wise average scores
- Lists students who did not submit
- Lists students scoring below 5
- Generates a JSON summary report

## Run Command

python task_2/src/main.py task_2/data/submissions.csv task_2/output/summary.json

## Output

The generated report is saved in:

task_2/output/summary.json
