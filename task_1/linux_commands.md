# Linux Commands Documentation

## pwd

Command:

```bash
pwd
```

Purpose:
Displays the current directory.

---

## ls

Command:

```bash
ls
```

Purpose:
Lists files and folders.

---

## mkdir

Command:

```bash
mkdir test_folder
```

Purpose:
Creates a new folder.

---

## cd

Command:

```bash
cd test_folder
```

Purpose:
Changes directory.

---

## touch

Command:

```bash
touch file.txt
```

Purpose:
Creates a file.

---

## cat

Command:

```bash
cat file.txt
```

Purpose:
Displays file content.

---

## cp

Command:

```bash
cp file.txt backup.txt
```

Purpose:
Copies files.

---

## mv

Command:

```bash
mv backup.txt newfile.txt
```

Purpose:
Moves or renames files.

---

## rm

Command:

```bash
rm newfile.txt
```

Purpose:
Deletes files.

---

## grep

Command:

```bash
grep hello file.txt
```

Purpose:
Searches text.

---

## find

Command:

```bash
find . -name "*.txt"
```

Purpose:
Finds files.

---

## head

Command:

```bash
head file.txt
```

Purpose:
Shows first lines.

---

## tail

Command:

```bash
tail file.txt
```

Purpose:
Shows last lines.

---

## wc

Command:

```bash
wc file.txt
```

Purpose:
Counts lines, words, characters.

---

## chmod

Command:

```bash
chmod +x script.sh
```

Purpose:
Changes permissions.
